# NoFlake

Collect refundable deposits via a shareable link. Reduce no-shows.

## Quick Start

### 1. Install dependencies

```bash
bundle install
```

### 2. Configure environment

```bash
cp .env.example .env
```

Fill in your Stripe keys in `.env`:

| Variable | Where to find it |
|---|---|
| `STRIPE_SECRET_KEY` | Stripe Dashboard → API keys (test mode) |
| `STRIPE_PUBLISHABLE_KEY` | Same page |
| `STRIPE_CLIENT_ID` | Stripe Connect settings → OAuth |
| `STRIPE_WEBHOOK_SECRET` | Generated by `stripe listen` (see below) |

### 3. Set up the database

```bash
rails db:create db:migrate
```

### 4. Start the app

```bash
bin/dev
```

App runs at **http://localhost:3002**

### 5. Start the Stripe webhook listener (separate terminal)

```bash
stripe listen --forward-to localhost:3002/webhooks/stripe
```

Copy the `whsec_...` secret it prints and put it in `.env` as `STRIPE_WEBHOOK_SECRET`.

## Full Flow (test mode)

1. Sign up at `/users/sign_up`
2. Click **Connect Stripe** on the dashboard and complete the OAuth flow with a test account
3. Create a booking link (service name + deposit amount in cents, e.g. 2500 = $25)
4. Copy the `/b/:slug` URL and open it in a new tab (or incognito)
5. Click **Reserve Spot** and use Stripe test card `4242 4242 4242 4242`
6. Return to dashboard — the payment appears in the table
7. Click **Refund** to return the deposit

## Architecture

```
app/
  controllers/
    application_controller.rb     # authenticate_user! default
    dashboard_controller.rb       # main dashboard
    booking_links_controller.rb   # create booking links
    bookings_controller.rb        # public /b/:slug + Stripe Checkout
    payments_controller.rb        # refund action
    stripe_connect_controller.rb  # Stripe Connect OAuth
    webhooks_controller.rb        # Stripe webhook handler
  models/
    user.rb          # Devise + stripe_connected?
    booking_link.rb  # slug generation, validations
    payment.rb       # paid / refunded states
```

## Routes

| Method | Path | Description |
|---|---|---|
| GET | `/dashboard` | Main dashboard |
| POST | `/booking_links` | Create booking link |
| GET | `/stripe/connect` | Start Stripe OAuth |
| GET | `/stripe/callback` | Stripe OAuth callback |
| GET | `/b/:slug` | Public booking page |
| POST | `/b/:slug/checkout` | Start Stripe Checkout |
| GET | `/success` | Post-payment success page |
| POST | `/payments/:id/refund` | Refund a deposit |
| POST | `/webhooks/stripe` | Stripe webhook receiver |
